# Task CRUD Application (Flask + MongoDB Atlas)

This is a simple **Task CRUD (Create, Read, Update, Delete)** backend application built using **Flask** and **MongoDB Atlas**.  
It provides REST APIs to manage tasks and is designed to work with a frontend (React or any UI).

## Features

- Create a new task
- Get all tasks
- Update a task
- Delete a task
- MongoDB Atlas cloud database
- CORS enabled (frontend ready)

## Tech Stack

- **Backend:** Flask (Python)
- **Database:** MongoDB Atlas (Cloud)
- **ORM:** PyMongo
- **Others:** dotenv, flask-cors

## Prerequisites

Make sure you have the following installed:

- Python **3.10+**
- pip
- MongoDB Atlas account


## MongoDB Atlas Setup

1. Create a **MongoDB Atlas account**
2. Create a **Cluster**
3. Create a **Database User**
4. Add IP Address:  0.0.0.0/0
